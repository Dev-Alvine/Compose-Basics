# Compose-Basics
A Simple App built in Jetpack Compose that shows some of its basics

## Features
#### 1. Material Design
##### shape, color, typography
#### 2. Compose Layouts
##### LazyColumn, Row, Column
#### 3. Lists and Animations, state
